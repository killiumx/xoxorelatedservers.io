Interior shells are the in-game layout used for the property. The shell is chosen independently of the zone — a property in an expensive area can still use a starter shell (Tier 1), and a budget-zone property can have a luxury shell (higher tier). Shell costs are fixed and added on top of the base property price.

💡

Zone ≠ Tier. Zone determines the location multiplier on the base price. Tier reflects how the property looks — the shell chosen. A house in Richman using Shell #9 is still a Tier 1 property, just in a luxury zone.
